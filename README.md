# 🍡 MOCHIMAN Website

The official website for **$MOCHIMAN** - The cutest mochi meme coin on Solana.

## Features

✨ **Modern Design** - Sleek dark theme with neon green accents
🎨 **Responsive** - Works on all devices
🚀 **Fast** - Optimized performance
💚 **Community Focused** - Built for the Mochi Army

## Getting Started

1. Clone this repository
2. Add your `mochiman.png` image to the root directory
3. Open `index.html` in your browser

## Sections

- **Hero** - Intro with mascot and call-to-action
- **About** - What is Mochiman
- **Why Mochiman** - Key features and benefits
- **Tokenomics** - Supply and tax information
- **Roadmap** - Phase 1, 2, and 3 plans
- **Community** - Links to join the Mochi Army
- **Footer** - Brand messaging

## Customization

Update these sections in `index.html`:
- Links in the buttons (`BUY NOW`, `JOIN TELEGRAM`)
- Telegram/Twitter/Pump.fun links in Community section
- Community stats and information
- Colors and styling in the `<style>` section

## Colors Used

- **Primary Green**: `#00ff88`
- **Secondary Blue**: `#4f8cff`
- **Background**: `#0b0b0b`
- **Text**: `#ffffff` & `#cccccc`

## Requirements

- A `mochiman.png` image file (420px or larger)
- A modern web browser

Enjoy! 🍡✨
